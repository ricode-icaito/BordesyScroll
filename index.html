-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- CSS (Cascading Style Sheets) es un lenguaje de programación que se usa para definir cómo se ven los elementos de una página web. Se utiliza para dar formato al HTML, definiendo atributos como el color, la forma, y el diseño de los elementos. CSS permite separar el contenido de una página web de su presentación, lo que facilita la creación y mantenimiento de sitios web. Con CSS, se pueden aplicar estilos a múltiples páginas al mismo tiempo, lo que ahorra tiempo y esfuerzo en el diseño web. HTML (Hypertext Markup Language) es el lenguaje de marcado utilizado para crear la estructura y el contenido de una página web. HTML define los elementos que componen una página, como encabezados, párrafos, imágenes y enlaces. HTML es el esqueleto de una página web, mientras que CSS se encarga de darle vida y estilo a ese esqueleto. Juntos, HTML y CSS forman la base del diseño web moderno.
